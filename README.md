# California Prop 13 Property Tax Disparity Analysis

Analyzes the property tax gap created by Proposition 13 (1978) across all 58 California counties. Produces interactive choropleth maps, bar charts, and scatter plots showing which counties have the largest disparities between assessed values and current market values — and who benefits most.

## Quick Start

```bash
# Set up environment
uv venv && uv pip install -e .

# Fetch raw data (saved to data/raw/ for reproducibility)
.venv/bin/python scripts/fetch_census_data.py
.venv/bin/python scripts/fetch_zillow_data.py

# Run all notebooks in order (generates data/processed/county_prop13.csv)
# Output must write back to the notebook files so generate_html.py can extract chart data
.venv/bin/python -m papermill notebooks/01_data_collection.ipynb notebooks/01_data_collection.ipynb --cwd notebooks
.venv/bin/python -m papermill notebooks/02_assessment_ratio_map.ipynb notebooks/02_assessment_ratio_map.ipynb --cwd notebooks
.venv/bin/python -m papermill notebooks/03_top_bottom_counties.ipynb notebooks/03_top_bottom_counties.ipynb --cwd notebooks
.venv/bin/python -m papermill notebooks/04_revenue_gap_map.ipynb notebooks/04_revenue_gap_map.ipynb --cwd notebooks
.venv/bin/python -m papermill notebooks/05_income_correlation.ipynb notebooks/05_income_correlation.ipynb --cwd notebooks

# Generate the final HTML report
.venv/bin/python scripts/generate_html.py
```

Output: `docs/report.html` — a single self-contained file with all interactive Plotly charts.

## Methodology

**Assessment Ratio = Tax-Derived Residential AV / Census Aggregate Market Value**

- **Assessed value (numerator)**: Census B25090 (aggregate property taxes paid by mortgage holders) / 0.011 effective tax rate. This is directly observed from tax returns, avoiding the need to estimate what fraction of the county's total assessed roll is residential.
- **Market value (denominator)**: Census B25082 (aggregate value of owner-occupied housing units) — a direct survey-based sum of self-reported home values.
- Lower ratio = larger Prop 13 benefit (long-tenure owners, high-appreciation areas).

**Annual Tax Gap = (Market Value - Assessed Value) x 1.1%**

- 1.1% = California's 1% base rate plus average local bonds/special assessments.
- Represents estimated additional revenue if all homes were assessed at current market value.

## Data Sources

| Source | Variable | Description |
|--------|----------|-------------|
| Census ACS 5-Year 2022 | B25082 | Aggregate value of owner-occupied housing units |
| Census ACS 5-Year 2022 | B25090 | Aggregate real estate taxes paid (mortgage holders) |
| Census ACS 5-Year 2022 | B25003 | Owner-occupied housing unit count |
| Census ACS 5-Year 2022 | B19013 | Median household income |
| CA BOE Annual Report 2022-23 | — | Total secured assessed values by county (fallback) |
| Zillow Research ZHVI | — | Median home values, SFR+condo, all tiers (fallback) |

No API keys required. Census API works unauthenticated for these queries.

## Project Structure

```
src/ca_data.py                  # Data fetching, fallback data, metric computation
scripts/fetch_census_data.py    # Fetch Census data -> data/raw/census_acs_2022.csv
scripts/fetch_zillow_data.py    # Fetch Zillow data -> data/raw/zillow_zhvi.csv
scripts/generate_html.py        # Combine notebook outputs -> docs/report.html
notebooks/01_data_collection    # Fetch/merge all sources -> data/processed/county_prop13.csv
notebooks/02_assessment_ratio_map  # Choropleth: assessment ratio by county
notebooks/03_top_bottom_counties   # Bar charts: all counties ranked
notebooks/04_revenue_gap_map       # Choropleth: tax gap by county
notebooks/05_income_correlation    # Scatter: income vs Prop 13 benefit
docs/report.html                # Final self-contained interactive report
```

## Notebook Pipeline

Notebooks must run in order — 02-05 depend on `data/processed/county_prop13.csv` generated by 01.

1. **01_data_collection** — Fetches and merges BOE, Zillow, and Census data, computes tax metrics, saves merged CSV.
2. **02_assessment_ratio_map** — Choropleth map of assessment ratios (lower = more Prop 13 benefit).
3. **03_top_bottom_counties** — Bar charts ranking all 58 counties by assessment ratio and gap per household.
4. **04_revenue_gap_map** — Choropleth maps of total and per-household annual tax gap.
5. **05_income_correlation** — Scatter plots exploring whether income correlates with Prop 13 benefit.
